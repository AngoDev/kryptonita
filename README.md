# Kryptonita

Kryptonita is a Ruby gem that provides a lot of functions for hashing, encrypt and decrypt

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kryptonita'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kryptonita
